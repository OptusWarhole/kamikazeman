Etape 1: Lancer le serveur. (port par défaut 4242)
cd server
./server

Etape 2: vérifier son ip (ifconfig, en0 sur mac)

Etape 3: Ouvrir bomberman.xcodeproj

Etape 4: Run (le jeu commence à partir de 4 joueurs connectés, il faut donc faire 4 fois Run depuis le meme projet xcode)

Etape 5: Une fois sur la fenetre de jeu, appuyez sur X, puis commencez a tapé l'ip suivi du port, séparé par des espaces (pas de points, que des espaces)
192 [Espace] XXX [Espace] XXX [Espace] XXX [Espace] 4242 

Etape 6: 4 client connectés on peut commencer à jouer (HAUT BAS GAUCHE DROITE , ESPACE POUR POSER UNE BOMBE)
